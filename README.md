# Automated File Sorter

A Python script that automatically organises files in a directory 
into subfolders based on file type.

## What it does
- Creates subfolders for CSV files, image files, and text files
- Scans a directory and moves each file into the correct subfolder
- Skips files that have already been moved

## Libraries used
- `os` — for reading directory contents and creating folders
- `shutil` — for moving files

## How to use
1. Open the notebook
2. Change the `path` variable to your own folder directory
3. Run all cells

## Skills demonstrated
File system automation, conditional logic, and directory management
